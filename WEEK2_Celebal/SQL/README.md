# Celebal Technologies Summer Internship 2026

## Week 2 Task: Relational E-Commerce Database Design & Analysis (ShopEase)

### 📌 Project Overview
This repository contains the full production-ready delivery for the Week 2 database engineering task of the Celebal Summer Internship. The core objective of this project is to implement a robust relational schema for **ShopEase**—a mid-sized e-commerce business operating across India—to help the management team extract vital operational insights regarding customer demographics, sales pipelines, inventory status, and transactional integrity.
